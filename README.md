# 玩法
一種卡牌戰鬥遊戲。
玩家選擇一個`角色`，並進入`戰鬥`。
遊戲目標是使用卡牌擊敗敵人。
在五場`戰鬥`後，玩家會面對一個強大的`敵人`，擊敗它就算贏得遊戲。
# 戰鬥
戰鬥開始前會洗牌。
敵人在畫面的右半邊登場，並顯示`生命值`。
角色在畫面的左半邊登場，並顯示`生命值`。
一場`戰鬥`可能會有一或多個`敵人`，`敵人`會同時登場，玩家需要同時對付他們。
一場`戰鬥`由數個`回合`組成，由玩家的`回合`開始遊戲，然後是`敵人`的`回合`，交替進行。
`敵人`會在畫面上顯示下一個`回合`的行動模式，玩家可以根據這些信息來決定自己的策略。
失敗的條件是`角色`的`生命值`歸零，勝利的條件是所有`敵人`的`生命值`歸零。
失敗的優先度高於勝利，當`角色`和`敵人`同時死亡時，玩家會被判定為失敗。
# 回合
每個玩家的`回合`開始時，玩家從`牌組`抽5張`牌`，並且`能量`被重置為3點。
在每個`回合`，玩家可以使用被抽到的`牌`一次，並消耗對應的`能量`。使用後的牌會被放入`棄牌堆`。
玩家可以在任何時候選擇結束回合。
玩家的`回合`結束後，未使用的`牌`會被放入`棄牌堆`。
`敵人`的`回合`開始後，`敵人`根據其`行動模式`執行相應的行動。
如果有多個敵人，則按照它們在畫面上的順序依次執行行動。
已`死亡`的`敵人`不會執行行動。
`敵人`的`回合`結束後，玩家的下一個`回合`開始。
# 角色
一個具有特定`最大生命值`、`能力`、`初始牌組`的遊戲替身。
# 敵人
一個具有特定`最大生命值`、`行動模式`的遊戲替身。
敵人只會傷害玩家，不會傷害其他敵人。
敵人可能會有特殊的`能力`，這些能力會在特定的條件下觸發特定的效果。
# 行動模式
`敵人`在每個`回合`會根據其`行動模式`執行相應的行動。
`行動模式`是一個列表，其結構為action_pattern[turn][action_index]，其中`turn`表示回合數，`action_index`表示該回合的行動索引。turn 從0開始，當達到列表末尾時會從頭開始。
```python
action_pattern = [
    [
        {'type': 'attack', 'damage': 5},
        {'type': 'defend', 'block': 3}
    ],
    [
        {'type': 'attack', 'damage': 10}
    ],
    [
        {'type': 'defend', 'block': 5}
    ]
]
```
每個`回合`是一個`行動列表`；每個`行動元素`是一個`字典`，包含 type 與參數。
敵人的行動目標根據行動類型決定，可能是：
1. 玩家
2. 自身（通常是防禦行動）
3. 同伴（如果有多個敵人）
傷害行動只能對玩家造成傷害，不能對其他敵人造成傷害。
敵人在同個回合可能會有多個行動，這些行動會按照列表中的順序依次執行。
`defend`與其他非傷害效果可作用於`自身`或`同伴`。
# 生命值
一組數字，用來表示`角色`還能受到多少`傷害`。
生命值為零的時候會被判定為`死亡`。
生命值不可以超過`最大生命值`。
生命值不可以為負數。
# 最大生命值
一組數字，用來表示`角色`能夠持有的最大的`生命值`。
# 死亡
`角色`的`生命值`為零的時候會被判定為`死亡`。
`敵人`的`生命值`為零的時候會被判定為`死亡`。
# 牌組
`牌組`由`牌`組成，決定玩家在戰鬥中可能抽到哪些`牌`。
如果`牌組`剩下的牌少於玩家需要抽出的`牌`，則全部抽出剩下的`牌`，把`棄牌堆`的`牌`放入`牌組`並洗牌，抽牌抽到所需的數量。
# 棄牌堆
使用後的牌會進入`棄牌堆`。
# 卡牌
`牌`分做三種，`攻擊牌`、`效果牌`、與`能力牌`。
能夠直接讓`敵人`減少`生命值`的牌叫做`攻擊牌`；
能夠在`角色`附加能力的叫做`能力牌`；
能夠在`角色`或`敵人`身上產生`狀態`或附加`格擋`的叫做`效果牌`。
使用`牌`需要消費`能量`，有的時候也需要消費`生命值`。
如果使用`牌`時`角色`的`生命值`不足以支付`牌`的生命值消耗，則角色`死亡`。
此`死亡`同樣遵守失敗優先於勝利的規則。
使用範圍根據卡牌描述決定，可能是：
1. 單一敵人
2. 所有敵人
3. 自己
如果一張牌有多個效果，處理的順序為：
1. 消耗`能量`和`生命值`
2. 執行`牌`的效果
# 能量
`角色`需要使用`能量`才能使用`牌`，如果未持有足夠的`能量`，不能使用`牌`。
請注意，持有零`能量`的時候可以使用消耗`能量`為零的`牌`。
`敵人`不需要`能量`來執行行動。
# 傷害
`傷害`能夠減少被作用對象的`生命值`。
# 格擋
`格擋`能夠減少`傷害`造成的效果，比例為1:1。一點`格擋`能夠減少一點`傷害`。
玩家的格擋在每個玩家回合開始時重置為零。
敵人的格擋在每個敵人回合開始時重置為零。
# 狀態
`狀態`是一種特殊的效果，能夠在`角色`或`敵人`身上持續一段時間。
根據卡牌描述，觸發時間點可能是：
1. 玩家的回合開始
2. 敵人的回合開始
# 能力
`能力`是一種特殊的效果，在本場戰鬥中持續存在，並且在特定的條件下觸發特定的效果。
# Framework
1. Python for backend
2. Pygame for GUI

# Settings
角色內容在`characters.yaml`中定義。
卡牌內容在`cards.yaml`中定義。
敵人內容在`enemies.yaml`中定義。
環境設定在`settings.yaml`中定義。
狀態內容在`statuses.yaml`中定義。
